# 🌾 Pixel Farm - 像素农场

A Stardew Valley-inspired farming game built with pure HTML5 Canvas + JavaScript.

**[Play Now →](https://120424304-dev.github.io/pixel-farm-game/)**

![Game Screenshot](screenshot.png)

## 🎮 Features

- **Grid-based farming system** - Plant, water, and harvest crops
- **Time system** - Day/night cycle with seasons
- **Multiple tools** - Hoe, seeds, watering can, harvest basket
- **Crop growth stages** - Watch your plants grow from seed to harvest
- **Save/Load system** - LocalStorage saves your progress
- **Mobile support** - Touch controls for on-the-go farming

## 🚀 Play Locally

```bash
git clone https://github.com/120424304-dev/pixel-farm-game.git
cd pixel-farm-game
open index.html
```

Or simply open `index.html` in your browser.

## 🎯 How to Play

### Controls (Desktop)
- **WASD / Arrow Keys** - Move
- **SPACE / E** - Use tool
- **1-5** - Select tool

### Controls (Mobile)
- **D-Pad** - Move
- **A Button** - Use tool

### Tools
| Tool | Emoji | Function |
|------|-------|----------|
| Hand | 🤚 | Basic interaction |
| Hoe | ⛏️ | Till grass into soil |
| Seeds | 🌱 | Plant crops |
| Water | 💧 | Water crops |
| Harvest | 🧺 | Harvest mature crops |

### Gameplay Loop
1. Use **Hoe** on grass to create farmland
2. Plant **Seeds** on tilled soil
3. **Water** your crops daily
4. Wait for crops to grow (2-5 days)
5. **Harvest** and sell for profit!

## 🛠️ Tech Stack

- HTML5 Canvas - Rendering
- Vanilla JavaScript - Game logic
- LocalStorage - Save system
- CSS3 - UI styling
